What are arrays?

An array is a special variable, which can hold more than one value at a time

var car1= "Benz";

var car2= "Fiat";

var car3= "BMW";


// what happens when you want to store 100 cars? combine them into an array

var cars = ["Benz", "Fiat", "BMW"];

Using an array literal is the easiest way to create a JavaScript Array

// var array_name = [item1, item2, ...];

var cars = ["Benz", "Fiat", "BMW"];

Accessing Array Elements

Now you know how to create an array, but lets think about accessing the variables stored in an array. You can access element using array indexes.

Each array element has an index associated to it. Indexes are numbers that are similar to an address of a house in the same street.

var cars = ["Benz", "Fiat", "BMW"];


// To access the third element of the above array

var thirdElement = cars[2]; // variable thirdElement would now store "BMW"

  • What! shouldn't it be cars[3] to access the third element? Why cars[2]?
  • Because, Array indexes are Zero-based
  • Indexes of arrays start from 0, so the first element is stored in 0th index.
  • To access the first element in an array m is, m[0]
  • To access the last element in an array m that has n number of elements is, m[n-1]

Array Operations

There are several handy array operations in JS

var cars = ["Benz", "Fiat", "BMW"];


// Add an item to the end of an Array - push(newElement)

cars.push("Toyota"); // output:- cars = ["Benz", "Fiat", "BMW", "Toyota"]


// Remove an item from the end of an Array - pop()

cars.pop(); // output:- cars = ["Benz", "Fiat", "BMW"]


// Remove an item from the beginning of an Array - shift()

cars.shift(); // output:- cars = ["Fiat", "BMW"]


// Add an item to the beginning of an Array - unshift(newElement)

cars.unshift("Porsche"); // output:- cars = ["Porsche", "Fiat", "BMW"]


// Find the index of an item in the Array - unshift(elementValue)

var index = cars.indexOf("Porsche"); // output:- index = 0


// Remove an item by index position - splice(index, 1)

var removedItem = cars.splice(index, 1);

// output:- removedItem = ["Porsche"]; cars=["Fiat", "BMW"]

Iterating over an array

Using a simple for-loop you can loop through the elements of an array

var cars = ["Benz", "Fiat", "BMW"];


for(int index = 0; index <= cars; index++){
  console.log(cars[index]);
}

This simple loop will print all the elements of the array to the console. You can perform additional logical operations inside the loop to apply the logic to each and every element.

// alternative approch with for-each method

var cars = ["Benz", "Fiat", "BMW"];


cars.forEach(function(car){
 console.log((car))
})
